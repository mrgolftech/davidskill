# 职场关系

## 核心认知

职场人际关系和私人关系最大的区别：**利益结构更明确、权力不对等更显著、你不能选择"不交往"。** 所以职场关系的核心技能是：在权力和利益的框架下，找到合作共赢的方式。

## 上下级关系

### 与上级相处

**核心原则**：理解领导的压力和诉求。领导不是敌人——他也有他的 KPI、他的上级、他的焦虑。

**向上管理的实用策略**

| 场景 | 策略 |
|------|------|
| 不合理的任务安排 | 不要直接说"做不了"，说"我梳理了一下优先级，A和B可能冲突，您看先做哪个" |
| 被抢功劳 | 在汇报中自然地体现你的贡献（"关于XX方案，我在调研阶段发现了…"） |
| 领导批评你 | 先认错再解释（"确实是我没做好，原因是…，下次我会…"） |
| 想提加薪/升职 | 用业绩说话，提前准备好你的贡献清单 |
| 领导风格你不喜欢 | 适应他的风格，不要期待他改。微信型就微信汇报，邮件型就发邮件 |

**和领导意见不一致**
- 不要在公开场合反驳领导
- 私下沟通，用"我有个不太成熟的想法…"开头
- 如果领导坚持，执行他的方案。执行出了问题不要说"我早就说了"
- 如果涉及原则或法律底线，书面记录，必要时跳过他向上反映

**新领导上任**
- 前三个月是关键期——这是他对你形成第一印象的时候
- 主动汇报工作进展，让他了解你在做什么
- 不要在新领导面前说前领导的坏话

### 作为领导

**核心原则**：下属需要的是清晰的方向 + 必要的支持 + 公平的评价。

**实用建议**
- 批评私下说、表扬公开说
- 分配任务要说清楚期望和标准，不要让下属猜
- 护短但不纵容——对外维护团队，对内严格要求
- 不要微管理——告诉他要什么结果，别管他怎么做

## 同事关系

### 核心认知

同事不是朋友，但也不是敌人。最好的同事关系是：**工作上配合默契，私下保持友善但有边界。**

### 常见困境与策略

**同事甩锅**
- 第一次可以不计较，但要确保有书面记录（邮件、聊天记录）
- 如果是常态：在协作开始时就明确分工，书面确认
- 被甩锅后不要在背后抱怨，在汇报时用事实说话

**办公室小团体**
- 不主动加入也不主动对抗
- 和所有人保持友善但不过度亲密
- 不参与背后议论任何人——今天他和你说别人坏话，明天就和别人说你坏话

**同事升职你没有**
- 先控制情绪，不要表现出明显的不满
- 客观分析差距在哪，是能力、资历还是关系
- 真诚恭喜对方——这是成年人的体面，也是投资
- 如果觉得不公平，可以找领导了解情况（"我想知道我还有哪些需要提升的"）

**工作中的竞争**
- 良性竞争：互相激励、各凭本事
- 恶性竞争：使绊子、背后捅刀
- 应对恶性竞争：不要以牙还牙，把精力放在做出成绩上；留好证据；必要时向领导反映

**同事请你帮忙**
- 分清"帮忙"和"帮他干活"——前者可以，后者不行
- 如果经常被人当免费劳动力：学会说"我手上这个比较急，你看能不能找XX"
- 帮忙后不要到处说——但对方应该知道你帮了他

## 师徒/带教关系

### 作为新人/徒弟

**核心原则**：姿态要低但不卑微，学习要主动但不烦人。

- 刚进来少说多看多做——先搞清楚这个团队的潜规则
- 有问题先自己想、查资料，实在不行再问（不要什么都问）
- 师傅教你的时候认真记，不要让他教第二遍
- 师傅发脾气：如果是为工作，虚心接受；如果是无理取闹，忍一时，找机会和领导反映
- **感恩的表达**：逢年过节一条问候消息、偶尔请师傅吃个饭——这些小事但很有用

### 作为带教/师傅

- 教人是投资——你带出来的人以后可能成为你的助力
- 对笨的人要有耐心：分析是理解力的问题还是操作熟练度的问题，分别对症
- 对调皮的人：先建立信任再提要求，讲道理比发脾气有效
- 不要藏私——"教会徒弟饿死师傅"是过时的思维

## 职场社交

### 饭局/应酬

**基本原则**
- 座次：面对门的是主位，左尊右卑，主宾坐主位右手边
- 敬酒：先敬领导/长辈/客人，碰杯时你的杯子要低于对方
- 话题：不谈政治、不谈宗教、不说别人坏话

**不想喝酒的应对**
- "我过敏/在吃药/开车来的"——这些借口比"我不想喝"好用
- 如果被逼：以茶代酒，态度诚恳

**饭局中的社交**
- 不要只和认识的人说话——这是拓展人脉的机会
- 注意听别人说话，找到共同话题
- 不要抢话、不要只谈自己

### 离职时的处理

- 至少提前一个月正式沟通（不是突然甩一封离职信）
- 做好交接——你的职业口碑比这份工作更值钱
- 不要在走的时候说坏话——行业圈子很小
- 和前同事保持友善联系——以后可能会再合作
