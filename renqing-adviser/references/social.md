# 社交关系

## 亲戚关系

### 核心认知

亲戚关系是中国人际关系中最"身不由己"的一种——你没法选择亲戚，但必须应对。核心策略：**热情有度、帮忙有界、不卑不亢。**

### 常见困境与策略

**亲戚催婚/催生/催二胎**
- 本质：他们焦虑的不是你的幸福，是他们的面子和"正常标准"
- **应对**：
  - 不要正面对抗（"关你什么事"只会升级矛盾）
  - 用模糊回应消解："在看呢"、"缘分到了自然就有了"、"我也着急呢"
  - 转移话题到对方身上："您家XX最近怎么样？"
  - 实在烦了，让父母帮你挡

**亲戚求你办事**
- **可以帮的**：用心尽力，让亲戚满意
- **不能帮的**：不要硬着头皮答应，解释清楚为什么帮不了
  - "这个事不在我的权限范围内"
  - "我打听过了，现在政策不允许"
  - "我能力有限，怕帮倒忙"
- **违反原则的**：坚决拒绝，但态度温和
  - "这个真不行，查出来我们都有问题"

**亲戚借钱**
- 比朋友借钱更棘手——因为还多了一层亲情绑架
- **应对原则**和朋友借钱类似，但多一条：
  - 如果是长辈借，金额不大，可以当作孝敬，别指望还
  - 如果金额大，一定要说清楚期限，最好打借条

**节日走亲戚**
- 带礼物是基本礼仪，不在贵在心意
- 了解对方的忌讳（有的人忌"钟/梨/伞"谐音不吉利）
- 到了先问候长辈，给小孩红包要提前准备
- 不要空手去，不要空手回（对方给你带的东西记得带走）

**亲戚之间的攀比**
- 不主动炫耀（房子、车子、孩子成绩）
- 别人炫耀时微笑点头就好，不必接话
- 被问到敏感话题（收入、房价）可以打哈哈："一般般啦""够吃够喝"

## 邻里关系

### 核心认知

邻里是长期关系——你们可能要做好多年邻居。所以处理邻里问题要留余地，不要把关系搞僵。

### 常见困境与策略

**噪音问题**
- 第一次：善意提醒（"不好意思打扰一下，最近晚上声音有点大，可能影响到孩子休息了"）
- 多次无效：书面沟通或通过物业/居委会
- 最后手段：报警或走法律途径
- 不要以噪音还击——这只会螺旋升级

**公共区域占用**
- 先友善沟通，大多数人不是故意的
- 如果不改善，通过物业或业委会处理
- 不要自己去移动别人的东西——容易引发正面冲突

**邻居帮忙与人情**
- 收了邻居的水果/食物，记得回礼
- 帮忙看个快递、浇个花这种小事要主动
- 不要过度借用邻居的东西（尤其是钱和车）

## 社交场合

### 送礼

**核心原则**：送礼的本质是表达心意，而不是交易。

| 场景 | 注意事项 |
|------|---------|
| 拜访长辈 | 实用为主（水果、保健品、茶叶），了解对方有什么忌口 |
| 看望病人 | 鲜花、水果、营养品。不送有数字"4"的东西 |
| 婚礼/满月 | 看当地标准，随大流。关系近的可以多给一点 |
| 乔迁 | 实用家居用品、绿植。不送刀（断交之意）、不送钟（送终谐音） |
| 商务送礼 | 不要太贵（有行贿嫌疑），不要太便宜（显得敷衍）。包装要好 |
| 回礼 | 收到礼物后找机会回礼，价值略高于收到的 |

**不要送的东西**
- 钟表（送终）、伞（散）、梨（离）、鞋（邪）——虽然年轻人不太在意，但送长辈要注意
- 白色或黑色的花（不吉利联想）
- 太贵重的东西（让人有压力）

### 饭局社交

**被邀请**
- 确认场合性质（纯社交还是有事要谈）
- 准时到——不要最晚来
- 不熟的人多听少说，找到共同话题再展开
- 结束时表达感谢

**你请客**
- 提前了解客人的饮食禁忌
- 点菜比人数多 2-3 个就够，不用铺张
- 主动安排座位（不要让客人尴尬地站着不知道坐哪）
- 结账时不要当着客人面看账单

### 寒暄与闲聊

**安全话题**：天气、美食、旅行、孩子教育（对方有孩子的话）
**危险话题**：收入、房价、政治、宗教、对方的感情状况
**万能开场**："最近忙什么呢？"、"上次你说的XX怎么样了？"

### 拒绝邀请

- "不好意思那天刚好有安排了"（不需要说具体安排）
- "下次一定！"（如果你确实想去但这次不行）
- 不要说"我再看看"然后不回复——这比直接拒绝更让人不舒服

## 陌生人互动

### 核心原则

和陌生人的互动虽然短暂，但也是人际修养的体现。

**被占便宜（插队、侵占空间等）**
- 小事：评估值不值得较真，很多时候"算了"是最优解
- 大事：冷静、清晰地表达你的立场，不需要发火

**服务纠纷**
- 先礼后兵：先正常沟通，不行再找管理者
- 不要对服务人员发火——他们通常不是决策者
- 有理有据比大声喊叫有效得多

**被搭讪/被骗**
- 信任你的直觉——如果感觉不对，就是不对
- 不必因为"不好意思拒绝"而给陌生人你的个人信息
- 涉及金钱的一律拒绝
