# 朋友关系

## 核心认知

朋友关系是所有人际关系中最"自愿"的——不像家人由血缘决定，不像同事由工作决定。正因为自愿，它也最脆弱：维护靠双方主动，一旦不维护就自然消散。

## 朋友的层级

不是所有朋友都用同一种方式相处：

| 类型 | 特征 | 相处原则 |
|------|------|---------|
| **点头之交** | 认识但不深交 | 保持礼貌和善意就够了 |
| **普通朋友** | 偶尔联系、聚聚 | 人情往来、量力帮忙 |
| **好朋友** | 互相信任、经常沟通 | 真诚相待、有话直说 |
| **挚友/密友** | 可以交心、无条件信任 | 像家人一样，但比家人多了选择的成分 |
| **诤友** | 敢于指出你的问题 | 最珍贵的一类，虚心接受批评 |

## 常见困境与策略

### 朋友借钱

这是中国人际关系中最常见也最棘手的场景之一。

**判断框架**
- 金额大小：小钱（一顿饭钱）直接借无需纠结；大钱要慎重
- 对方信用：以前借的还了没？
- 借的原因：急事（看病、应急）vs 消费（买车、炒股）
- 你的承受能力：这笔钱丢了对你有多大影响

**借的策略**
- 借之前心里把这笔钱当作"可能丢了"——如果不能接受就别借
- 大额要打借条，写明金额和还款时间。"亲兄弟明算账"是保护友谊
- 不要因为面子借超出自己能力的钱

**不借的策略**
- 不要说"我没钱"（对方可能知道你有）
- 可以说："最近我也有大笔支出/在还贷/家里有事，实在周转不开"
- 如果金额合理但你不太放心，可以借一小部分："我也紧，先借你 X，剩下的你看看别的渠道"

**催还的策略**
- 到期不还：先旁敲侧击（"上次那笔钱你方便的时候…"）
- 多次不还：直接说（"我最近也需要用钱，之前借你的能还一下吗"）
- 赖账不认：衡量金额和友谊，小钱就当买了个教训，大钱走法律途径

### 帮忙与人情

**核心原则**：力所能及就帮，超出能力坦诚拒绝。不要勉强自己帮忙然后心里怨恨。

**拒绝帮忙的得体方式**
- 不要说"不想帮"，说"帮不了"
- 给出具体原因（即使是编的也比直接拒绝好）
- 如果可能，提供替代方案（"我帮不了这个，但你可以找XXX试试"）

**被帮忙后的回馈**
- 口头感谢是最低限度
- 找机会回馈（不一定等价，重要的是有这个心）
- 不要让别人的帮助变成理所当然

### 朋友之间的地位变化

人们的生活轨迹不同，朋友之间出现收入差距、社会地位差距是必然的。

**你混得好，朋友一般**
- 不要炫耀、不要摆架子
- 主动买单但不要表现得理所当然
- 聊天时注意话题不要都围绕自己的成就

**朋友混得好，你一般**
- 真心为朋友高兴（这很难但很重要）
- 不要因为自卑就疏远
- 不要因为对方条件好就频繁求帮忙

### 信任破裂

**秘密被泄露**
- 先确认：是故意的还是无心的？
- 如果是无心的：可以表达不满但给机会
- 如果是故意的：降低这个人在你心里的信任等级

**背后说坏话**
- 先从多个来源确认（传话容易变形）
- 如果确认了：可以直接找对方问（"我听说你和XX说了我什么什么，是真的吗"）
- 不要以牙还牙——你也背后说他，只会让关系螺旋下降

### 异性朋友的边界

**有伴侣的情况下和异性朋友相处**
- 让你的伴侣也认识这个朋友——透明是最好的信任建设
- 不要单独深夜聊天、不要聊感情话题
- 如果朋友表白了：明确拒绝，不暧昧

**对方表白了但你只想做朋友**
- 直接但温和地表明："我很珍惜我们的友谊，但只是朋友的感觉"
- 不要为了不伤害对方而暧昧回应——那样伤害更大
- 给彼此一段距离，等感情消退后再恢复正常来往

## 维护友谊的实用建议

- **主动联系**：不要等对方来找你，关系是需要主动维护的
- **记住重要日子**：生日、考试、面试——在这些节点上发个消息
- **少评判、多倾听**：朋友找你倾诉时，他需要的是一个听众而不是法官
- **允许疏远**：有些朋友走散了是正常的，不必强行维持
