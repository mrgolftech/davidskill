# 关系类型速查表

根据用户描述的场景，判断属于以下哪种关系类型，然后调取对应知识库。

## 家庭关系 → family.md

| 关系 | 典型场景 | 核心难点 |
|-----|---------|---------|
| 夫妻 | 争吵、沟通障碍、家务分配、育儿分歧、经济纠纷 | 期望差异、沟通不足 |
| 婆媳 | 干涉生活、育儿观念冲突、生活习惯差异 | 边界模糊、权力争夺 |
| 亲子 | 代沟、管教方式、经济独立、赡养老人 | 控制与自主的平衡 |
| 兄弟姐妹 | 分家、财产、赡养分摊、偏心 | 利益分配、公平感 |
| 姑嫂/妯娌 | 家庭聚会尴尬、背后议论、资源竞争 | 间接竞争、面子 |

## 恋爱关系 → romance.md

| 关系 | 典型场景 | 核心难点 |
|-----|---------|---------|
| 择偶 | 相亲、条件评估、家人反对 | 现实与理想的冲突 |
| 表白/追求 | 如何开口、被拒绝、暧昧不明 | 面子风险、信号误读 |
| 恋爱中 | 磨合、争吵、异地、对方家庭 | 期望管理、价值观差异 |
| 失恋 | 分手痛苦、挽回、前任纠缠 | 情绪管理、自我修复 |

## 朋友关系 → friends.md

| 关系 | 典型场景 | 核心难点 |
|-----|---------|---------|
| 普通朋友 | 借钱、帮忙、人情往来 | 边界感、经济清明 |
| 密友 | 信任破裂、秘密泄露、价值观分歧 | 信任的脆弱性 |
| 同学 | 聚会、攀比、地位变化后的疏远 | 身份认同变化 |
| 异性朋友 | 被误解、对方表白、伴侣不满 | 友谊与爱情的边界 |

## 职场关系 → workplace.md

| 关系 | 典型场景 | 核心难点 |
|-----|---------|---------|
| 上下级 | 不合理安排、抢功劳、被穿小鞋、向上管理 | 权力不对等 |
| 同事 | 甩锅、竞争、合作摩擦、小团体 | 利益竞争、协作成本 |
| 师徒/带教 | 不愿教、教了不学、满师后关系 | 知识权力、感恩与独立 |
| 客户/合作方 | 被刁难、催款、谈判 | 利益博弈、面子工程 |

## 社交关系 → social.md

| 关系 | 典型场景 | 核心难点 |
|-----|---------|---------|
| 亲戚 | 催婚、借钱、求办事、节日往来 | 人情债、边界模糊 |
| 邻里 | 噪音、占道、宠物纠纷 | 长期共处、不好翻脸 |
| 陌生人 | 被占便宜、插队、服务纠纷 | 短期博弈、情绪管理 |
| 社交场合 | 饭局、敬酒、寒暄、送礼 | 礼仪规范、面子文化 |

## 判断原则

- 优先判断**核心关系**（谁和谁之间的问题）
- 如果涉及多重关系，识别**主要矛盾**在哪对关系上
- 如果不确定，宁可多读几个知识库文件也不要漏掉
