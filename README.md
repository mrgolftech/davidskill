# davidskill

胖胖的 AI Skill & 常用工具分享仓库。

大家好，我是胖胖，一个工作了十几年的中年老登，也是一个四岁半女儿的父亲。公众号「**慢半拍的胖胖**」，写 AI、科技、教育、育儿——记录 AI 如何改变我的工作方式，以及我对科技和教育的诚实观察。

这个仓库里放的是我自己日常在用的 Hermes agent skill 和各种小工具。从协议发现到脚本落地，再到包装成 agent 可调用的 skill，每一个都是真正跑在生产环境里的东西。希望能给你一些启发。

欢迎关注我的公众号，一起交流 AI 时代的工具哲学和创作心得。
